# FinTrack

## Introduction

FinTrack is a personal expense tracking web application built with React, Vite, and Supabase. It helps users manage their daily expenses, monitor spending habits, and gain insights through an intuitive dashboard and visual analytics.

The project was developed to simplify personal finance management while providing a modern and responsive user experience.

## Live Demo

https://fintrack-app-nu.vercel.app

## Features

* User authentication using Supabase
* Secure login and signup functionality
* Add, edit, and delete expenses
* Categorize transactions for better organization
* Monthly budget tracking
* Interactive expense analytics and charts
* Responsive design for desktop and mobile devices

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/FinTrack.git
cd FinTrack
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
